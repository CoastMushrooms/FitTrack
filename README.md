# FitTrack

This repository contains files for the FitTrack App, which takes in the user's height and weight and gives them their health status.

## Prerequisites

- Java JDK (version 8 or higher) - Download from [Oracle's website](https://www.oracle.com/java/technologies/javase-downloads.html) or use your system's package manager (e.g., `apt install openjdk-11-jdk` on Ubuntu).

## Installation and Running

1. Clone the repository:
   ```
   git clone https://github.com/CoastMushrooms/FitTrack.git
   ```

2. Navigate to the project directory:
   ```
   cd FitTrack
   ```

3. Compile the Java code:
   ```
   javac Main.java
   ```

4. Run the application:
   ```
   java Main
   ```

The application will launch a GUI window for tracking fitness activities.
